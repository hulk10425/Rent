# Livegether iOS


>為了解決FB上租屋社團找室友的資訊過於雜亂，因而開發了Livegether App，
使用者能夠刊登找室友資訊以及透過多人聊天室即時對話，提高找室友的效率


##Feature
<ul>
  <li>Home
  <ul type=circle>
    <li>Homemates List
    <li>Multiple Chat
    <li>Detail homemates infomation
    <li>Add homemates condition
    <li>Sort Homemates List
  </ul>
</ul>

<ul>
<li>Messages
  <ul type=circle>
    <li>Newest message list
  </ul>
</ul>
<ul>
<li>personal setting
  <ul type=circle>

    <li>Push notification setting

  </ul>
</ul>
